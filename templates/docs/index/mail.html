  1. Open app/mail.php
  2. The MVC framework uses PHPmailer.
  3. Select option of Mailer. By default it is set to SMTP using gmail.
  4. Set the credentials of username and password. Set the From emailaddress and FromName
  5. If you have planned for POP3 set it to true in the array with other credentials.
