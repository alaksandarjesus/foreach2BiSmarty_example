Enter the database credentials at app/database.php.

This mvc support multiple database. You are free to add multiple database to app/database.php

  1. Create a file as MymodelModel.php in model folder.
  2. Open your controller file and call the model as Mymodel::method();
  3. Get Data From Database.

    4. General Syntax

    Database::get(App::DB('dbparams'),column,table,condition=optional,data=optional)

    Get All Data from the table

    Database::get(App::DB('dbparams'),*,table);

    Output

    Array("Column1"=>Array("data1","data2"),"Column2"=>Array("data3","data4")....)

    Get All Data from the table with condition

    Database::get(App::DB('dbparams'),*,table,"WHERE ColumnName = ?",array(value1));

    Output

    Array("Column1"=>Array("data1","data2"),"Column2"=>Array("data3","data4")....);

    Get Specific Column from the table with condition

    Database::get(App::DB('dbparams'),"Column1",table,"WHERE ColumnName = ?",array(value1));

    Output

    Array("Column1"=>Array("data1","data2"....))

  4. Insert Into Database.

    5. General Syntax

    Database::insert(App::DB('dbparams'),table,array("column1"=>"value","column2"=>"value"));

  5. Update Database.

    6. General Syntax

    Database::update(App::DB('dbparams'),table,ColumnName,ColumnValue, array(ColumnName,'=',ColumnValue));

  6. Delete From Database.

    7. General Syntax

    Database::delete(App::DB('dbparams'),table,"WHERE ColumnName = ? ",array("value"));
